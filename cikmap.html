Finding Company CIKs¶

The secedgar.utils.get_cik_map function is provided as a utility to easily fetch CIKs based on a company’s ticker or name.

secedgar.utils.get_cik_map(key='ticker')¶

Get dictionary of tickers to CIK numbers.

Parameters

key (str) – Should be either “ticker” or “title”. Choosing “ticker” will give dict with tickers as keys. Choosing “title” will use company name as keys.

Returns

Dictionary with either ticker or company name as keys, depending on key argument, and corresponding CIK as values.

New in version 0.1.6.

By default, get_cik_map fetches a dictionary using company tickers as keys.

Note

In both examples below, only the first 5 results are shown.

In [1]: from secedgar.utils import get_cik_map

In [2]: dict(list(get_cik_map().items())[:5])
Out[2]: 
{'AAPL': '320193',
 'MSFT': '789019',
 'AMZN': '1018724',
 'GOOG': '1652044',
 'TSLA': '1318605'}

To get a dictionary with the company names as the keys, use key="title".

In [3]: from secedgar.utils import get_cik_map

In [4]: dict(list(get_cik_map(key="title").items())[:5])
Out[4]: 
{'Apple Inc.': '320193',
 'MICROSOFT CORP': '789019',
 'AMAZON COM INC': '1018724',
 'Alphabet Inc.': '1652044',
 'Tesla, Inc.': '1318605'}
